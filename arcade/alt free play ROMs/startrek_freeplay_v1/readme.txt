Star Trek Freeplay patch

Description
-----------
This patch allows free play for Star Trek.  

Press start button to start a game for that number of players

Instructions
------------

1) Remove the EPROM on the CPU board (the one the large harness plugs into)
2) Load the rom you have as a file using an EPROM burner.
3) Download a patcher program
4) Run the patcher program against the rom data with the patch file
5) Burn the new data to an EPROM and install
